Simple Twitter Widget for Wordpress

== Installation ==

1. Upload `wptwitterplugin` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the Simple Twitter Widget on your sidebar.
